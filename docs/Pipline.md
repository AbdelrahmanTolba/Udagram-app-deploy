# Pipeline Process

The pipeline is setup and connected with this GitHub repository in CircleCI.

## Order of commands

1. Install Node, the EB CLI and the AWS CLI.
2. FrontEnd & BackEnd install
3. FrontEnd & BackEnd build
4. FrontEnd & BackEnd deploy
